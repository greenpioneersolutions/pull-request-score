## Writing a custom metric plugin

You can extend the metrics library by registering your own plugins. Each plugin implements the `MetricPlugin` interface and is loaded via `register`.

```ts
import type { MetricPlugin } from 'pull-request-score'
import { register } from 'pull-request-score'

const filesChanged: MetricPlugin = {
  slug: 'files_changed',
  description: 'Number of files changed in a PR',
  calculate(pr) {
    return pr.changedFiles
  },
}

register(filesChanged)
```
